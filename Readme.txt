Para ejecutar el proyecto, ir a la ubicación en donde se encuentra y escribir cmd en la "dirección" para que se abra la consola, posteriormente para su visualización y ejecución escribir dentro del cmd: ionic serve --lab , al esperar que esto cargue, en el navegador predeterminado se abrirá una pestaña mostrando la app donde podrá interactuar con las distintas funciones disponibles. Se instaló el plugin Ionic Storage como base de datos, cabe destacar que sin este plugin no funcionara correctamente la App.
